      \ body: the department, agency, board or commission responsible for making the\
      \ described resource publicly available.\nguidance:\n  Contact information should\
      \ be included for all new information resources that are added to the Government\
      \ Data Catalogue. Generally, `contactName` will be a support or branch unit\
      \ that will either respond to the user or refer the inquiry to a subject matter\
      \ expert.\n  \n  Because the Government Data Catalogue may include historical\
      \ resources for which there is no longer a suitable contact point, this element\
      \ has not been made mandatory.\n  \n  When a `contactName` is provided for an\
